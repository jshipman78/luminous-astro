# Luminous Productions Website

Website for Luminous Productions, a digital media services company based in Commerce, TX.

**Live Site:** [luminouspro.com](https://luminouspro.com)

## About

Luminous Productions offers photography, web design, and business technology services to Northeast Texas since 2001.

**Services:**
- Real Estate Photography (HDR photos, Matterport 3D tours, drone aerials, video)
- Portrait Photography (weddings, family, seniors, headshots)
- Digital Services (web design, business automation, video production)

## Tech Stack

- **Framework:** [Astro](https://astro.build)
- **Styling:** [Tailwind CSS](https://tailwindcss.com)
- **Fonts:** DM Sans, Fraunces (via Fontsource)
- **Hosting:** Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Development server runs at `http://localhost:4321`

## Project Structure

```
luminous-astro/
├── public/          # Static assets (images, fonts, favicon)
├── src/
│   ├── components/  # Reusable Astro components
│   ├── layouts/     # Page layouts
│   └── pages/       # Site pages (file-based routing)
├── docs/            # Project documentation
│   └── SPEC.md      # Website specification
└── dist/            # Production build output
```
